# myqq
概述：程序设计实践（创）作业，编写了实现类似QQ功能的聊天服务器端代码和客户端代码。

服务器功能：监控客户端间的对话，可向某个或所有客户端广播，接收并处理客户端的注册、登录、添加好友等请求，并将依据处理结果对数据库进行操作。

客户端：注册时，得到服务器生成的随机账号，可以设置密码、其他个人信息并设置头像。登录后，可以与服务器对话，聊天室聊天，添加或删除好友、查看好友个人信息，修改自己的个人信息，更换头像和注销账号。

下载和安装注意事项：点击绿色的Code,选择Download Zip，即可下载完整文件，后续操作请详细阅读“mysql的配置和安装”、“服务器端代码的配置”和“客户端代码的配置”。
